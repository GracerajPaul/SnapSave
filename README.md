
# SnapSave - Elite Image Vault

SnapSave is a registration-free, PIN-based image vault web application designed for absolute privacy and security.

## Core Philosophy
Forging the future of digital sovereignty through uncompromising privacy. Developed under the leadership of Chairman Graceraj Paul.

## Features
- **Registration-Free**: No emails, no passwords. Access via Username and PIN.
- **Elite Security**: Client-side BCrypt hashing and secure Telegram-backed storage.
- **Auto-Deletion**: Configurable expiry (24h, 7d, 30d, Never) to maintain a clean digital footprint.
- **Emergency Lock**: Instantly disable vault access during security threats.
- **Bulk Export**: Download your entire vault as a secure ZIP archive.

## Tech Stack
- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Infrastructure**: Telegram Bot API for image hosting
- **Bundler**: Vite
- **Deployment**: Vercel

## Deployment
This project is optimized for **Vercel**. Simply connect your GitHub repository to Vercel and it will auto-deploy using the provided `vercel.json` and `package.json`.
